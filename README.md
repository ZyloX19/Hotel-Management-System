# Hotel Management System

## Overview
A full-stack web-based Hotel Management System built using **Python, Flask, and MySQL**.  
This system allows hotel staff to manage rooms, customer bookings, check-ins/check-outs, and view booking records efficiently.

---

## Technologies Used
- **Backend:** Python, Flask  
- **Database:** MySQL  
- **Frontend:** HTML  
- **Tools:** VS Code, MySQL Workbench  

---

## Features
- **User Authentication:** Secure login/logout system.  
- **Dashboard:** Central navigation to all modules.  
- **Room Management:** Add, view, and track room availability.  
- **Booking Module:** Book available rooms for customers with check-in/check-out dates.  
- **View Bookings:** Display all active bookings in a table.  
- **Check-out / Free Room:** Mark rooms as available after customer check-out while keeping historical booking records.  
- **Database Integration:** MySQL backend to store users, rooms, customers, and bookings.
